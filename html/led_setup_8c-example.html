led_setup.c

Enables a number of LEDs with a specified color and brightness

Parameters
nrNumber of LEDs ranging from 0 to 40
blParameter for holding blue value (0-255)
grParameter for holding green value (0-255)
rdParameter for holding red value (0-255)
lbParameter for holding brightness value (0-31)

Example how to use the led_setup function

led_setup(40,255,128,128,12)
